# 平台机制证据：给课程学员的阅读说明

先读这份本地证据摘要：

`skills/platform-positioning-benchmark/references/platform-mechanism-evidence.zh-CN.md`

它把小红书、抖音、哔哩哔哩、微博、视频号和公众号分成三类：官方可确认的机制、可在自己账号观察的指标、不能声称的“固定算法权重”。它是本课程使用的可读版本。

## 为什么有些官方 PDF 点开是空白

网信办算法备案站的原始附件存在动态访问风控。2026-07-17 复核时，部分深层 PDF 链接会在内嵌浏览器返回 `403` 或空白预览；这不表示 PDF 原文为空，也不表示该备案不存在。

课程交付保留原始 URL 供追溯，但不要求学员靠它完成学习。需要自行核验时，请从 [网信办算法备案查询入口](https://beian.cac.gov.cn/) 访问，并按算法名称搜索：

- 小红书：小红书个性化笔记推荐算法
- 抖音：抖音个性化推荐算法
- 哔哩哔哩：哔哩哔哩首页推荐算法
- 微博：微博信息流推荐算法

## 使用时的结论

- 可以说：平台公开说明会综合内容、用户行为、质量、多样性、去重或负反馈等信号。
- 不可以说：收藏、完播、转发、CPE 等某项指标具有固定的、适用于所有账号的推荐权重。
- 正确做法：把公开机制当作实验指标的候选，再用自己账号可见的数据验证内容定位与下一轮选题。
